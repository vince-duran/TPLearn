Text Colors:
- Entity Names: #0D47A1 (dark blue)
- Process Labels: #1B5E20 (dark green)
- Data Store Names: #E65100 (dark orange)
- Flow Labels: #424242 (dark gray)

STYLING EFFECTS:
Drop Shadows:
- Entities: 3px offset, #000000 20% opacity
- Processes: 2px offset, #000000 15% opacity
- Data Stores: 1px offset, #000000 10% opacity

Gradients:
- Entities: Linear gradient from light to medium blue
- Processes: Radial gradient from center light to edge medium green
- Data Stores: Linear gradient from light to medium yellow

Border Styles:
- Entities: Solid, 3px width, rounded corners
- Processes: Solid, 3px width, perfect circles
- Data Stores: Solid, 2px width, sharp corners (except missing right edge)

// ========================================
// 9. FINAL LAYOUT OPTIMIZATION
// ========================================

SPACING OPTIMIZATION:
Minimum Distances:
- Entity to Entity: 200px horizontal
- Process to Process: 180px horizontal, 160px vertical
- Data Store to Data Store: 200px horizontal, 70px vertical
- Entity to Process: 100px vertical
- Process to Data Store: 120px vertical

Flow Path Optimization:
1. Minimize crossing arrows
2. Use consistent curve radii (20px)
3. Group related flows with similar colors
4. Avoid sharp angle turns (< 45°)
5. Maintain parallel flows where possible

Visual Balance:
- Symmetrical layout with center alignment
- Equal weight distribution across quadrants
- Consistent element sizing within categories
- Harmonious color distribution

ACCESSIBILITY CONSIDERATIONS:
Color Contrast:
- All text must have 4.5:1 contrast ratio minimum
- Use patterns/textures in addition to colors
- Provide high-contrast alternative version

Font Legibility:
- Minimum 9pt font size for labels
- Clear, sans-serif fonts only
- Adequate white space around text
- No italic or condensed fonts for labels

// 10. QUALITY ASSURANCE CHECKLIST
// ========================================

ELEMENT VERIFICATION:
□ All 3 external entities present and properly positioned
□ All 8 core processes numbered correctly (1.0-8.0)
□ All 9 data stores labeled with correct IDs (D1-D9)
□ Entity rectangles have rounded corners and proper dimensions
□ Process circles are perfect circles with consistent sizing
□ Data store rectangles have open right edges

POSITIONING ACCURACY:
□ External entities aligned horizontally at top
□ Processes arranged in 2×4 grid formation
□ Data stores arranged in 3×3 grid formation
□ Adequate spacing between all elements
□ No overlapping elements or text
□ Symmetrical layout maintained

FLOW CONNECTIONS:
□ All major input flows from entities to processes
□ All major output flows from processes to entities
□ All data write flows from processes to stores
□ All data read flows from stores to processes
□ Arrow directions correct for all flows
□ Flow labels positioned clearly

COLOR AND STYLING:
□ Consistent color scheme applied
□ Proper contrast ratios maintained
□ Drop shadows applied appropriately
□ Border styles consistent within categories
□ Text colors match element categories

TEXT FORMATTING:
□ All entity names in uppercase and bold
□ Process numbers and names properly formatted
□ Data store IDs and names correctly labeled
□ Flow labels readable and positioned well
□ No text overlapping with other elements
□ Consistent font sizes and weights

FINAL REVIEW:
□ Overall layout is balanced and professional
□ Information hierarchy is clear
□ All elements are properly aligned
□ Canvas size accommodates all content
□ Legend or title block included if required
□ Ready for presentation or documentation use
